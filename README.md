# RESTful Node.js API Demo

### `Install`
 
Run 
```sh 
npm install
```

Run 
```sh
npm run dev
```

The server will run on port 8000

### `Dependencies`

```js
  "dependencies": {
    "body-parser": "^1.18.2",
    "express": "^4.16.3",
    "mongodb": "^3.0.7"
  },
  "devDependencies": {
    "nodemon": "^1.17.3"
  }
```

### `MongoDB setup `
Mongo database setup is through mLab.

### `HTTP method` 

* `GET`: Fetch data
* `POST`: Adds new data
* `PUT`: Updates data
* `DELETE`: Removes data
